Gambaran Umum Cloud Computing

Cloud computing :
-	Gabungan pemanfaatan teknologi komputer  dan pengembangan berbasis Internet yang tidak melibatkan user.  
-	Merupakan abstraksi dari infrastruktur kompleks yang disembunyikannya, suatu metoda komputasi di mana kapabilitas terkait teknologi informasi disajikan sebagai suatu layanan (as a service), sehingga pengguna dapat mengaksesnya lewat Internet tanpa mengetahui apa yang ada didalamnya. 
-	Informasi tersimpan permanen di server internet dan tersimpan secara sementara di komputer pengguna (client) termasuk di dalamnya adalah desktop, komputer tablet, notebook, komputer dinding, handheld, sensor-sensor, monitor 
Manfaat cloud computing :
•	Skalabilitas, menambah kapasitas penyimpanan data tanpa beli peralatan tambahan, misalnya hardisk dll. 
•	Aksesibilitas, akses data bisa dilakukan kapanpun dan dimanapun menggunakan internet.
•	Keamanan, keamanan data dijamin oleh penyedia layanan cloud computing, selain itu bisamengurangi biaya yang diperlukan untuk mengamankan data perusahaan.
•	Kreasi, user bisa melakukan/mengembangkan kreasi atau project melalui penyedia layanan cloud computing.
•	Kecemasan, data tersimpan aman di cloud meskipun hardisk atau gadget rusak.

Layanan Cloud Computing :
•	Infrastructure as a Service (IaaS), adalah layanan komputasi awan yang menyediakan infrastruktur IT berupa CPU, RAM, storage, bandwith dan konfigurasi lain. Tidak perlu komputer fisik untuk menginstal sistem operasi atau aplikasi. Contoh penyedia :  Amazon EC2, Telkom Cloud dan BizNetCloud.
•	Platform as a Service (PaaS), Adalah layanan yang menyediakan sistem operasi, database, web server dan framework aplikasi. Semua layanan ditanggung oleh penyedia layanan seperti  Amazon Web Service dan Windows Azure.
•	Software as a Service (SaaS), adalah layanan komputasi awan dimana user bisa langsung menggunakan aplikasi yang telah disediakan, seperti layanan e-mail dan media sosial.
•	Mobile "backend" as a service (MBaaS), layanan berbasis cloud untuk menghubungkan aplikasi mobile dengan penyimpanan cloud yang dijembatani oleh API (Application Programming Interface) dan SDK (Software Development Kit).. Jenis layanan berupa  user management, push notifications, integrasi dengan sosial network
•	Serverless computing, adalah model eksekusi kode komputasi awan di mana sepenuhnya dikelola oleh penyedia sesuai permintaan dan ditagihkan ke client.
•	Function as a service (FaaS), Sebuah layanan hosting yang memanfaatkan komputasi tanpa server 

Cara Kerja Cloud Computing :
-	Komputer lokal hanya melakukan installasi operating system pada satu aplikasi di internet. 
-	Semua aplikasi dijalankan oleh server, mulai dari E-mail, Pengolah kata, sampai program analisis data yang kompleks. 
-	Internet Protokol (IP) dapat digunakan untuk menetapkan dimana pengguna berada (geolocation). 
-	Domain Name System (DNS) mengarahkan pengguna ke sebuah cluster server yang dekat dengan pengguna sehingga situs bisa diakses dengan. 
-	Pengguna tidak login ke server, tetapi login ke layanan mereka menggunakan id sesi atau cookie. 
-	Webservers menjalankan perangkat lunak dan menyajikan pengguna dengan cara interface yang digunakan untuk mengumpulkan perintah atau instruksi dari pengguna (klik, mengetik, upload dan lain-lain) 
-	Web service telah memberikan mekanisme umum untuk pengiriman layanan, hal ini membuat Service-Oriented Architecture (SOA) ideal untuk diterapkan. 

 Yang diperlukan dalam implementasi cloud computing :
•	Computer front end, merupakan komputer desktop biasa.
•	Computer back end, berupa server komputer yang dilengkapi dengan data center dalam rak-rak.
•	Penghubung antara keduanya, penghubung keduanya bisa berupa jaringan LAN atau internet.

Penggunaan cloud computing dapat ditemui di Google Drive, Windows Azure.
